Banco de Dados
O projeto utiliza SQLite com as seguintes tabelas:

API Endpoints
POST /api/auth/login - Autenticação
GET /api/users - Listar usuários
POST /api/beneficiarios - Cadastrar beneficiário
GET /api/doadores - Listar doadores
